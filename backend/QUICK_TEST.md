# Быстрый запуск тестов

## Самый простой способ - Ручной тест

```bash
# 1. Убедитесь, что MongoDB и Redis запущены
# Проверка MongoDB:
mongosh --eval "db.adminCommand('ping')"

# Проверка Redis:
redis-cli ping
# Должен вернуть: PONG

# 2. Запустите ручной тест
cd backend
pnpm test:manual
```

Этот тест автоматически:
- ✅ Создает тестовые данные
- ✅ Выполняет полный цикл аукциона
- ✅ Проверяет все этапы
- ✅ Очищает данные после теста

## Автоматические тесты

```bash
cd backend

# Все тесты
pnpm test

# Конкретный тест
pnpm test auctionFullCycle
pnpm test bidService
pnpm test settlementService
```

## Что проверяется

### ✅ Полный цикл аукциона
- Создание → Ставки → Settlement → Следующий раунд → Завершение

### ✅ Размещение ставок
- Успешная ставка
- Предотвращение дубликатов
- Проверка баланса
- Заморозка средств

### ✅ Settlement
- Определение победителей
- Списание/возврат средств
- Идемпотентность

### ✅ Edge Cases
- Недостаточный баланс
- Tie-breaking
- Пустые ставки
- Отсутствие подарков

## Результат

Если все тесты прошли успешно, вы увидите:
```
✓ All tests passed successfully!
```

Если есть ошибки, они будут показаны с детальной информацией.
